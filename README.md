# miniBlog
php+mysql搭建的独立博客。

![blog][1]

# 功能

* 博客分页
* 文章评论
* 文章分类
* 友情链接
* 文章归档
* 关于页面
* 页面使用markdown语法

# 用于

* php初体验
* php操作mysql练习
* HTML/css入门练习
* SEO练习

# 要求环境

配置PHP相关环境&mysql

#使用说明

1）可作为PHP搭建独立博客的练习

2）可购买虚拟空间进行网站部署，注意修改config文件
